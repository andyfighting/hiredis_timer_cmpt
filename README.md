# hiredis_timer_cmpt
> An redis timer task component based on Hiredis source code development.

# Compile:
```Bash
       cd hiredis_timer_cmpt 
       make redis_timer
```
 
# How to use:
```Bash
      ./redis_timer -h
      eg: ./redis_timer -i 192.168.0.100 -p 6339 -a 123456
```
